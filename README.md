# GSGWR
This repository fully implements the **GSGWR model**, including distance matrix generation scripts, optimized large-sample model code, standard model code for small and medium samples, and supports five standard research datasets.

## Repository File Structure
```
Root Directory
├─ Data/ # Store all raw datasets and generated distance matrices
│  ├─ Housing.csv
│  ├─ zn.csv
│  ├─ HIV.csv
│  ├─ Crime.csv
│  ├─ Depression.csv
│  ├─ Geographical_distance_matrix.csv    # Spatial distance matrix generated by preprocessing scripts
│  └─ Similarity_distance_matrix.csv       # Attribute similarity distance matrix generated by preprocessing scripts
└─ GSGWR/ # All executable code stored in this folder
   ├─ GSGWR_Housing.py # GSGWR for large datasets (chunked memory loading, Numba acceleration, four-stage bandwidth search)
   ├─ GSGWR_main.py # Standard GSGWR for small & medium datasets (sample size < 3000)
   ├─ Preprocess_GeoMatrix.py # Script to generate geographical Euclidean distance matrix
   └─ Preprocess_SimMatrix.py # Script to generate attribute similarity distance matrix
```

## Complete Workflow
### Step 1: Generate two distance matrix files
1. Open `Preprocess_GeoMatrix.py`, modify the parameter `main_file = '../Data/xxx.csv'` in the configuration section to your target dataset name, then run the script. The spatial distance matrix will be saved to the Data folder.
2. Open `Preprocess_SimMatrix.py`, change `main_file` to the identical dataset name, then run the script. The attribute similarity distance matrix will be saved to the Data folder.

### Step 2: Run the corresponding model script
1. Small & medium datasets (zn / Crime / Depression / HIV): Run `GSGWR_main.py`
Modify the file paths of dataset and distance matrices inside the `load_all_data` function.
2. Large dataset (Housing): Run `GSGWR_Housing.py`
Two running modes are provided in the menu:
- 1 = Automatic bandwidth optimization search
- 2 = Fast fitting with fixed bandwidth

## Dataset Format Specification for Files in Data Folder
1. Column 1–4: Four coordinate variables
2. Column 5: Dependent variable Y
3. Column 6 and all subsequent columns: Independent explanatory variables X
